INTRODUCTION

Consequence is an interactive tool to raise awareness about the environment. Informs and makes visible the invisible. We fight for a better world changing the way of acting.

OBJECTIVE

The objective of this project was:
·Raise awareness about the veggie movement
·Engage the user through an interactive tool
·Disclose the movement

We had 4 points to cover:

ARCHITECTURE

We used a Branched/Parallel architecture for this web-app Separating the menu by different epics:
·Animals
·Products
·Additional information

WIREFRAMES

This wireframes helped us to start with the guerrilla testing to validate the functionalities and detect some issues with the interactions. After these testings we iterated the wireframes and added some colors and text style.
